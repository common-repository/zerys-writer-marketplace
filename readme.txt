== Description ==

Zerys is a project manage tool designed specifically for content projects. It also has a built-in writer marketplace that gives you access to thousands of professional, freelance writers. Use our award-winning process to plan your content strategy, and then get your content written by the best possible writer. First, discover the best page topics to write about. Then, use the title suggestion tool to build a list of compelling page titles. Only Zerys offers "SmartPost" reverse-bid technology, which guarantees you get the highest-rated writer, at the lowest possible price. Quickly sample writers until you find ones you love - then use our simple negotiation process to set a long-term rate for each writer on your team. Once you approve your content, auto-publish to Hubspot or Wordpress - or - distribute your content to thousands of sites out on the web to drive traffic. You only pay if you're 100% satisfied!


== Installation ==

To install, simply copy the 'zerys-writer-marketplace' folder to your 'wp-content/plugins/' directory. you must activate the plugin within the WordPress Site Admin section called 'Plugins'
